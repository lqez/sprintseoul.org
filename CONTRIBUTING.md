# Contributing

This doc describes how to build and develop this site

## Clone source

```
$ git clone git@github.com:sprintseoul/sprintseoul.org.git
$ cd sprintseoul.org
```


## Install modules

```
$ npm install
```


## Run development server

```
$ npm run develop
```


## Connect to the site

Run your browser and connect to <http://localhost:8000/>.
